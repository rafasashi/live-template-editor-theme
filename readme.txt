== D O C U M E N T A T I O N ==

Please, follow the steps below for a proper theme installation and use:

== Installation ==
1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.

== Menu Setup ==
1. In WordPress administration navigate to Appearance > Menus
2. Create a new menu by adding a "Menu Name" and saving the menu.
3. Proceed by adding Pages, Categories and Custom Links to the new menu.
4. Organize your menu by dragging and dropping the links to your desired locations.
5. Select menu location - ltple-theme supports 2 menus: top & footer.
6. Make sure to save your changes when you are done.

== Translation ==
This theme is translation/localization ready. File is located in 'ltple-theme/languages' folder.
1. Download poedit software here https://poedit.net/download. Install it and in this software translate .po file (line by line) screens. 
2. Once you have translated all the strings, you can save this as your .po file.  
If your translation is written in American English for example, your file name will look like en-US.po. 
Capitalization is important. For a full list of language and country codes, see these two links:
http://www.gnu.org/software/gettext/manual/gettext.html#Country-Codes
http://www.gnu.org/software/gettext/manual/gettext.html#Language-Codes
3. Once you save, POEdit by default automatically creates a .mo file alongside your .po file. Put these files into 'languages' folder.
4. Go to (WordPress Dashboard > Settings > General). Select your website langauge

== Logo Upload ==
Navigate to Appearance > Customize in your admin panel. There you will have the option to upload the logo.

== Footer Copyrights ==
Navigate to Appearance > Customize in your admin panel. There you will have the option to add your own copyright text which will appear in footer.

== Sticky Post ==
ltple-theme has a nice stylish sticky post area. If you want to use that area, first you would need to login to your WP-Admin Panel, and open up the respective post page. Look towards the right hand sidebar where you will see a Publish box. Look at the (Visibility: Public) field and click edit. Check the box to make the post Sticky and publish it. 

== Templates ==
ltple-theme has 2 more page templates besides the default one which has no sidebar: one with sidebar and one for developers. The last one can be used in example to set a homepage. A page template can activated by creating/editing a page , right area of the page editor -> dropdown select.
